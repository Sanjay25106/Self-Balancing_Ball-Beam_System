# Self-Balancing Ball-Beam System
Self-Balancing Ball-Beam System Using Camera-Based Vision Feedback

Abstract 


This project presents the design and implementation of a vision-based self-balancing ball-beam system. A standard ping-pong ball is stabilized at the center of a 50cm beam actuated at its midpoint by a servo motor. Real-time ball position is estimated from webcam frames using OpenCV-based image processing, and a PID controller computes the corrective beam angle. The control signal is relayed to an Arduino Mega via serial communication, which generates the corresponding PWM signal for the servo. The system demonstrates closed-loop feedback control integrating mechanical, electronic, and software subsystems, with potential applications in precision industrial automation and robotic stability control.
